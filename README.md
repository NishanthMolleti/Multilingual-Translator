# Multilingual-Translator
This project aims to convert text from one language to another 
This project uses huggingface model created by facebook (https://huggingface.co/facebook/m2m100_1.2B) to convert text from one language to another.
The source and result is limited to the languages listed in the above model's documentation i.e,
